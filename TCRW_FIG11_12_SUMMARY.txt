================================================================================
PHYSICS INTERPRETATION
================================================================================

BOUNDARY LOCALIZATION:
The chiral active particle (TCRW) transition matrix has a spectrum that
organizes around boundaries. Each boundary type (external edge, internal
defect interface) acts as a "sink" for certain eigenmodes.

EDGE MODES (Panel b, d):
- Eigenvalues with high boundary weight (red) are localized at interfaces
- These modes don't diffuse into the bulk
- Create the "edge current" observed in steady-state probability distributions
- Physical origin: chiral rotation + finite domain → prefer boundary motion

NESTED STRUCTURE (Panel d):
When multiple disconnected boundaries exist, the spectrum shows nested ovals.
This is because:
1. Each boundary has its own set of localized modes (one oval per boundary)
2. The eigenvalue densities for different boundaries slightly overlap
3. The color pattern (red/blue) reveals which boundary dominates each oval

This nested structure is a direct manifestation of bulk-boundary correspondence
in non-Hermitian topology. The interior defect boundaries carry modes that
would be delocalized in the interior (bulk states) but are confined at the
boundary-defect interface.
